Поиск уязвимостей в проектах на а Github

по языку программирования и шаблонам

Вы можете выбрать язык, на котором написаны программы, задать ключевое слово для отбора страниц и дополнительно шаблон для отбора внутри старниц

Обратите внимание, что обработка поиска может занять значательное время, особенно при использовании распространного ключевого слова,
поскольку GitHub ограничил количество запросов к серверам.
После каждого запроса страницы или файла требуется тайиаут (3 сек.)
Вполне вероятно, что на обработку вашего запроса понадобится несколько часов.
